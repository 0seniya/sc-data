3. Bījaṅgapañha

SC 1Ms 28Mi_1715SC 1Ms 28Mi_1715Msdiv 237“Bhante nāgasena, ‘bījassa dve aṅgāni gahetabbānī’ti yaṃ vadesi, katamāni tāni dve aṅgāni gahetabbānī”ti? “Yathā, mahārāja, bījaṃ appakampi samānaṃ bhaddake khette vuttaṃ deve sammā dhāraṃ pavecchante subahūni phalāni anudassati; evameva kho, mahārāja, yoginā yogāvacarena yathā paṭipāditaṃ sīlaṃ kevalaṃ sāmaññaphalamanudassati. Evaṃ sammā paṭipajjitabbaṃ. Idaṃ, mahārāja, bījassa paṭhamaṃ aṅgaṃ gahetabbaṃ.

SC 2SC 2Ms 28Mi_1716Puna caparaṃ, mahārāja, bījaṃ suparisodhite khette ropitaṃ khippameva saṃvirūhati; evameva kho, mahārāja, yoginā yogāvacarena mānasaṃ supariggahitaṃ suññāgāre parisodhitaṃ satipaṭṭhānakhettavare khittaṃ khippameva virūhati. Idaṃ, mahārāja, bījassa dutiyaṃ aṅgaṃ gahetabbaṃ. Bhāsitampetaṃ, mahārāja, therena anuruddhena—

SC 3Ms 28Mi_1717‘SC 3Ms 28Mi_1717‘Yathāpi khette parisuddhe,
bījañcassa patiṭṭhitaṃ;
Vipulaṃ tassa phalaṃ hoti,
api toseti kassakaṃ.

SC 4SC 4Ms 28Mi_1718Tatheva yogino cittaṃ,
suññāgāre visodhitaṃ;
Satipaṭṭhānakhettamhi,
khippameva virūhatī’”ti.

SC 5SC 5Ms 28Mi_1719Bījaṅgapañho tatiyo.
