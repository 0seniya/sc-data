413. Dhūmakārijātaka

SC 1PTS vp Pali 3.401Ms 22J_1631Msdiv 128“Rājā apucchi vidhuraṃ,
dhammakāmo yudhiṭṭhilo;
Api brāhmaṇa jānāsi,
ko eko bahu socati”.

SC 2Ms 22J_1632Msdiv 129“Brāhmaṇo ajayūthena,
pahūtedho vane vasaṃ;
Dhūmaṃ akāsi vāseṭṭho,
rattindivamatandito.

SC 3Ms 22J_1633Msdiv 130Tassa taṃ dhūmagandhena,
sarabhā makasaḍḍitā;
Vassāvāsaṃ upāgacchuṃ,
dhūmakārissa santike.

SC 4Ms 22J_1634Msdiv 131Sarabhesu manaṃ katvā,
ajā so nāvabujjhatha;
Āgacchantī vajantī vā,
tassa tā vinasuṃ ajā.

SC 5PTS vp Pali 3.402Ms 22J_1635Msdiv 132Sarabhā sarade kāle,
pahīnamakase vane;
Pāvisuṃ giriduggāni,
nadīnaṃ pabhavāni ca.

SC 6Ms 22J_1636Msdiv 133Sarabhe ca gate disvā,
ajā ca vibhavaṃ gatā;
Kiso ca vivaṇṇo cāsi,
paṇḍurogī ca brāhmaṇo.

SC 7Ms 22J_1637Msdiv 134Evaṃ yo saṃ niraṃkatvā,
āgantuṃ kurute piyaṃ;
So eko bahu socati,
dhūmakārīva brāhmaṇo”ti.

SC 8Ms 22J_1638Dhūmakārijātakaṃ aṭṭhamaṃ.
