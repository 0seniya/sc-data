423. Indriyajātaka

SC 1PTS vp Pali 3.464Ms 22J_1738Msdiv 60“Yo indriyānaṃ kāmena,
vasaṃ nārada gacchati;
So pariccajjubho loke,
jīvantova visussati.

SC 2Ms 22J_1739Msdiv 61Sukhassānantaraṃ dukkhaṃ,
dukkhassānantaraṃ sukhaṃ;
Sosi patto Sosi patto sukhā dukkhaṃ,
pāṭikaṅkha varaṃ sukhaṃ.

SC 3Ms 22J_1740Msdiv 62Kicchakāle kicchasaho,
yo kicchaṃ nātivattati;
Sa kicchantaṃ sukhaṃ dhīro,
yogaṃ samadhigacchati.

SC 4PTS vp Pali 3.466Ms 22J_1741Msdiv 63Na heva kāmāna kāmā,
nānatthā nātthakāraṇā;
Na katañca niraṅkatvā,
dhammā cavitumarahasi”.

SC 5Ms 22J_1742Msdiv 64“Dakkhaṃ gahapatī sādhu,
saṃvibhajjañca bhojanaṃ;
Ahāso atthalābhesu,
atthabyāpatti abyatho”.

SC 6Ms 22J_1743Msdiv 65“Ettāvatetaṃ paṇḍiccaṃ,
api so api so devilo bravi;
Na yito kiñci pāpiyo,
yo indriyānaṃ vasaṃ vaje”.

SC 7PTS vp Pali 3.468Ms 22J_1744Msdiv 66“Amittānaṃva hatthatthaṃ,
sivi pappoti māmiva;
Kammaṃ vijjañca dakkheyyaṃ,
vivāhaṃ sīlamaddavaṃ;
Ete ca yase hāpetvā,
nibbatto sehi kammehi.

SC 8Ms 22J_1745Msdiv 67Sohaṃ sahassajīnova,
abandhu aparāyaṇo;
Ariyadhammā apakkanto,
yathā peto tathevahaṃ.

SC 9Ms 22J_1746Msdiv 68Sukhakāme dukkhāpetvā,
āpannosmi padaṃ imaṃ;
So sukhaṃ nādhigacchāmi,
ṭhito bhāṇumatāmivā”ti.

SC 10Ms 22J_1747Indriyajātakaṃ sattamaṃ.
