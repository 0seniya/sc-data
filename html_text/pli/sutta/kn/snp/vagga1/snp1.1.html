• Sutta Nipāta 1.1

Uragasutta

SC 1SC Verse 1PTS Verse 1Ms 18Sn_2Yo uppatitaṃ vineti kodhaṃ,
Visaṭaṃ sappavisaṃva osadhehi;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 2SC Verse 2PTS Verse 2Ms 18Sn_3Yo rāgamudacchidā asesaṃ,
Bhisapupphaṃva saroruhaṃ vigayha;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 3SC Verse 3PTS Verse 3Ms 18Sn_4Yo taṇhamudacchidā asesaṃ,
Saritaṃ sīghasaraṃ visosayitvā;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 4SC Verse 4PTS Verse 4Ms 18Sn_5Yo mānamudabbadhī asesaṃ,
Naḷasetuṃva sudubbalaṃ mahogho;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 5SC Verse 5PTS Verse 5Ms 18Sn_6Yo nājjhagamā bhavesu sāraṃ,
Vicinaṃ pupphamivā udumbaresu;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 6SC Verse 6PTS Verse 6Ms 18Sn_7Yassantarato na santi kopā,
Itibhavābhavatañca vītivatto;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 7SC Verse 7PTS Verse 7Ms 18Sn_8Yassa vitakkā vidhūpitā,
Ajjhattaṃ suvikappitā asesā;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 8SC Verse 8PTS Verse 8Ms 18Sn_9Yo nāccasārī na paccasārī,
Sabbaṃ accagamā imaṃ papañcaṃ;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 9SC Verse 9PTS Verse 9Ms 18Sn_10Yo nāccasārī na paccasārī,
Sabbaṃ vitathamidanti ñatva loke;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 10SC Verse 10PTS Verse 10Ms 18Sn_11Yo nāccasārī na paccasārī,
Sabbaṃ vitathamidanti vītalobho;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 11SC Verse 11PTS Verse 11Ms 18Sn_12Yo nāccasārī na paccasārī,
Sabbaṃ vitathamidanti vītarāgo;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 12SC Verse 12PTS Verse 12Ms 18Sn_13Yo nāccasārī na paccasārī,
Sabbaṃ vitathamidanti vītadoso;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 13SC Verse 13PTS Verse 13Ms 18Sn_14Yo nāccasārī na paccasārī,
Sabbaṃ vitathamidanti vītamoho;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 14SC Verse 14PTS Verse 14Ms 18Sn_15Yassānusayā na santi keci,
Mūlā ca akusalā samūhatāse;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 15SC Verse 15PTS Verse 15Ms 18Sn_16Yassa darathajā na santi keci,
Oraṃ āgamanāya paccayāse;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 16SC Verse 16PTS Verse 16Ms 18Sn_17Yassa vanathajā na santi keci,
Vinibandhāya bhavāya hetukappā;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇaṃ.

SC 17SC Verse 17PTS Verse 17Ms 18Sn_18Yo nīvaraṇe pahāya pañca,
Anigho tiṇṇakathaṅkatho visallo;
So bhikkhu jahāti orapāraṃ,
Urago jiṇṇamivattacaṃ purāṇanti.

Uragasuttaṃ paṭhamaṃ.
