• Sutta Nipāta 2.13

Sammāparibbājanīyasutta

SC 1SC Verse 362PTS Verse 359PTS vp Pali 63Ms 18Sn_419“Pucchāmi muniṃ pahūtapaññaṃ,
Tiṇṇaṃ pāraṅgataṃ parinibbutaṃ ṭhitattaṃ;
Nikkhamma gharā panujja kāme,
Kathaṃ bhikkhu sammā so loke paribbajeyya”.

SC 2SC Verse 363PTS Verse 360Ms 18Sn_420“Yassa maṅgalā samūhatā,
(iti bhagavā):
Uppātā supinā ca lakkhaṇā ca;
So maṅgaladosavippahīno,
Sammā so loke paribbajeyya.

SC 3SC Verse 364PTS Verse 361Ms 18Sn_421Rāgaṃ vinayetha mānusesu,
Dibbesu kāmesu cāpi bhikkhu;
Atikkamma bhavaṃ samecca dhammaṃ,
Sammā so loke paribbajeyya.

SC 4SC Verse 365PTS Verse 362Ms 18Sn_422Vipiṭṭhikatvāna pesuṇāni,
Kodhaṃ kadarīyaṃ jaheyya bhikkhu;
Anurodhavirodhavippahīno,
Sammā so loke paribbajeyya.

SC 5SC Verse 366PTS Verse 363Ms 18Sn_423Hitvāna piyañca appiyañca,
Anupādāya anissito kuhiñci;
Saṃyojaniyehi vippamutto,
Sammā so loke paribbajeyya.

SC 6SC Verse 367PTS Verse 364Ms 18Sn_424Na so upadhīsu sārameti,
Ādānesu vineyya chandarāgaṃ;
So anissito anaññaneyyo,
Sammā so loke paribbajeyya.

SC 7SC Verse 368PTS Verse 365PTS vp Pali 64Ms 18Sn_425Vacasā manasā ca kammunā ca,
Aviruddho sammā viditvā dhammaṃ;
Nibbānapadābhipatthayāno,
Sammā so loke paribbajeyya.

SC 8SC Verse 369PTS Verse 366Ms 18Sn_426Yo vandati manti nunnameyya,nunnameyya,
Akkuṭṭhopi na sandhiyetha bhikkhu;
Laddhā parabhojanaṃ na majje,
Sammā so loke paribbajeyya.

SC 9SC Verse 370PTS Verse 367Ms 18Sn_427Lobhañca bhavañca vippahāya,
Virato chedanabandhanā ca bhikkhu;
So tiṇṇakathaṃkatho visallo,
Sammā so loke paribbajeyya.

SC 10SC Verse 371PTS Verse 368Ms 18Sn_428Sāruppaṃ attano viditvā,
No ca bhikkhu hiṃseyya kañci loke;
Yathātathiyaṃ viditvā dhammaṃ,
Sammā so loke paribbajeyya.

SC 11SC Verse 372PTS Verse 369Ms 18Sn_429Yassānusayā na santi keci,
Mūlā ca akusalā samūhatāse;
So nirāso nirāso anāsisāno,
Sammā so loke paribbajeyya.

SC 12SC Verse 373PTS Verse 370PTS vp Pali 65Ms 18Sn_430Āsavakhīṇo pahīnamāno,
Sabbaṃ rāgapathaṃ upātivatto;
Danto parinibbuto ṭhitatto,
Sammā so loke paribbajeyya.

SC 13SC Verse 374PTS Verse 371Ms 18Sn_431Saddho sutavā niyāmadassī,
Vaggagatesu na vaggasāri dhīro;
Lobhaṃ dosaṃ vineyya paṭighaṃ,
Sammā so loke paribbajeyya.

SC 14SC Verse 375PTS Verse 372Ms 18Sn_432Saṃsuddhajino vivaṭṭacchado,
Dhammesu vasī pāragū anejo;
Saṅkhāranirodhañāṇakusalo,
Sammā so loke paribbajeyya.

SC 15SC Verse 376PTS Verse 373Ms 18Sn_433Atītesu anāgatesu cāpi,
Kappātīto aticcasuddhipañño;
Sabbāyatanehi vippamutto,
Sammā so loke paribbajeyya.

SC 16SC Verse 377PTS Verse 374Ms 18Sn_434Aññāya padaṃ samecca dhammaṃ,
Vivaṭaṃ disvāna pahānamāsavānaṃ;
Sabbupadhīnaṃ parikkhayāno,
Sammā so loke paribbajeyya”.

SC 17SC Verse 378PTS Verse 375PTS vp Pali 66Ms 18Sn_435“Addhā hi bhagavā tatheva etaṃ,
Yo so evaṃvihārī danto bhikkhu;
Sabbasaṃyojanayogavītivatto,
Sammā so loke paribbajeyyā”ti.

Sammāparibbājanīyasuttaṃ terasamaṃ.
