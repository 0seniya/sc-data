Yasmiṃ samaye rūpūpapattiyā maggaṃ bhāveti vivicceva kāmehi … pe … paṭhamaṃ jhānaṃ upasampajja viharati vinīlakasaññāsahagataṃ … pe … vipubbakasaññāsahagataṃ … pe … vicchiddakasaññāsahagataṃ … pe … vikkhāyitakasaññāsahagataṃ … pe … vikkhittakasaññāsahagataṃ … pe … hatavikkhittakasaññāsahagataṃ … pe … lohitakasaññāsahagataṃ … pe … puḷavakasaññāsahagataṃ … pe … aṭṭhikasaññāsahagataṃ, tasmiṃ samaye phasso hoti … pe … avikkhepo hoti … pe … ime dhammā kusalā.
