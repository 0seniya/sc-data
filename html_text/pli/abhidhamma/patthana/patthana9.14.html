9.14. Kilesagocchaka, Kusalattika

SC 1Ms 40P9_169Msdiv 68Nokilesaṃ nakusalaṃ dhammaṃ paṭicca … pe … nanokilesaṃ nakusalaṃ dhammaṃ paṭicca …. (Saṃkhittaṃ.)

SC 2Ms 40P9_170Hetuyā nava … pe … vipāke ekaṃ … pe … avigate nava.

SC 3Ms 40P9_171Msdiv 69Nasaṃkilesikaṃ nakusalaṃ dhammaṃ paṭicca … pe … naasaṃkilesikaṃ nakusalaṃ dhammaṃ paṭicca …. (Saṃkhittaṃ.)

SC 4Ms 40P9_172Msdiv 70Nasaṃkiliṭṭhaṃ nakusalaṃ dhammaṃ paṭicca … pe … naasaṃkiliṭṭhaṃ nakusalaṃ dhammaṃ paṭicca …. (Saṃkhittaṃ.)

SC 5Ms 40P9_173Hetuyā pañca, ārammaṇe dve … pe … āsevane ekaṃ … pe … avigate pañca.

SC 6Ms 40P9_174Msdiv 71Nakilesasampayuttaṃ nakusalaṃ dhammaṃ paṭicca … pe … nakilesavippayuttaṃ nakusalaṃ dhammaṃ paṭicca …. (Saṃkhittaṃ.)

SC 7Ms 40P9_175Hetuyā pañca, ārammaṇe dve … pe … vipāke ekaṃ … pe … avigate pañca.

SC 8Ms 40P9_176Msdiv 72Nakilesañceva naasaṃkilesikañca nakusalaṃ dhammaṃ paṭicca … pe … naasaṃkilesikañceva nano ca kilesaṃ nakusalaṃ dhammaṃ paṭicca …. (Saṃkhittaṃ.)

SC 9Ms 40P9_177Hetuyā nava … pe … vipāke ekaṃ … pe … avigate nava.

SC 10Ms 40P9_178Msdiv 73Nakilesañceva naasaṃkiliṭṭhañca nakusalaṃ dhammaṃ paṭicca … pe … naasaṃkiliṭṭhañceva nano ca kilesaṃ nakusalaṃ dhammaṃ paṭicca … pe … nakilesañceva nakilesavippayuttañca nakusalaṃ dhammaṃ paṭicca … pe … nakilesavippayuttañceva nano ca kilesaṃ nakusalaṃ dhammaṃ paṭicca … pe …. (Sabbattha nava. Vipākaṃ natthi.)
