Rūpaṃkusalākusalantikathā

SC 1Ms 32Kv_2404Msdiv 760☑ Rūpaṃ kusalanti? Āmantā. Sārammaṇaṃ, atthi tassa āvaṭṭanā … pe … paṇidhīti? Na hevaṃ vattabbe … pe … nanu anārammaṇaṃ, natthi tassa āvaṭṭanā … pe … paṇidhīti? Āmantā. Hañci anārammaṇaṃ, natthi tassa āvaṭṭanā … pe … paṇidhi, no ca vata re vattabbe—“rūpaṃ kusalan”ti.

SC 2Ms 32Kv_2405Msdiv 761☑ Alobho kusalo sārammaṇo, atthi tassa āvaṭṭanā … pe … paṇidhīti? Āmantā. Rūpaṃ kusalaṃ sārammaṇaṃ, atthi tassa āvaṭṭanā … pe … paṇidhīti? Na hevaṃ vattabbe … pe ….

SC 3Ms 32Kv_2406☑ Adoso kusalo … pe … amoho kusalo … pe … saddhā … vīriyaṃ … sati … samādhi … pe … paññā kusalā sārammaṇā, atthi tāya āvaṭṭanā … pe … paṇidhīti? Āmantā. Rūpaṃ kusalaṃ sārammaṇaṃ, atthi tassa āvaṭṭanā … pe … paṇidhīti? Na hevaṃ vattabbe … pe ….

SC 4Ms 32Kv_2407☑ Rūpaṃ kusalaṃ anārammaṇaṃ, natthi tassa āvaṭṭanā … pe … paṇidhīti? Āmantā. Alobho kusalo anārammaṇo, natthi tassa āvaṭṭanā … pe … paṇidhīti? Na hevaṃ vattabbe … pe ….

SC 5Ms 32Kv_2408☑ Rūpaṃ kusalaṃ anārammaṇaṃ, natthi tassa āvaṭṭanā … pe … paṇidhīti? Āmantā. Adoso kusalo … pe … paññā kusalā anārammaṇā, natthi tāya āvaṭṭanā … pe … paṇidhīti? Na hevaṃ vattabbe … pe ….

SC 6Ms 32Kv_2409Msdiv 762☑ Rūpaṃ akusalanti? Āmantā. Sārammaṇaṃ, atthi tassa āvaṭṭanā … pe … paṇidhīti? Na hevaṃ vattabbe … pe … nanu anārammaṇaṃ, natthi tassa āvaṭṭanā … pe … paṇidhīti? Āmantā. Hañci anārammaṇaṃ, natthi tassa āvaṭṭanā … pe … paṇidhi, no ca vata re vattabbe—“rūpaṃ akusalan”ti … pe ….

SC 7Ms 32Kv_2410Msdiv 763☑ Lobho akusalo sārammaṇo, atthi tassa āvaṭṭanā … pe … paṇidhīti? Āmantā. Rūpaṃ akusalaṃ sārammaṇaṃ, atthi tassa āvaṭṭanā … pe … paṇidhīti? Na hevaṃ vattabbe … pe … doso … moho … māno … pe … anottappaṃ akusalaṃ sārammaṇaṃ, atthi tassa āvaṭṭanā … pe … paṇidhīti? Āmantā. Rūpaṃ akusalaṃ sārammaṇaṃ, atthi tassa āvaṭṭanā … pe … paṇidhīti? Na hevaṃ vattabbe … pe ….

SC 8Ms 32Kv_2411☑ Rūpaṃ akusalaṃ anārammaṇaṃ, natthi tassa āvaṭṭanā … pe … paṇidhīti? Āmantā. Lobho akusalo anārammaṇo, natthi tassa āvaṭṭanā … pe … paṇidhīti? Na hevaṃ vattabbe … pe … rūpaṃ akusalaṃ anārammaṇaṃ, natthi tassa āvaṭṭanā … pe … paṇidhīti? Āmantā. Doso … moho … pe … anottappaṃ akusalaṃ anārammaṇaṃ, natthi tassa āvaṭṭanā … pe … paṇidhīti? Na hevaṃ vattabbe … pe ….

SC 9Ms 32Kv_2412Msdiv 764☒ Na vattabbaṃ—“rūpaṃ kusalampi akusalampī”ti? Āmantā. Nanu kāyakammaṃ vacīkammaṃ kusalampi akusalampīti? Āmantā. Hañci kāyakammaṃ vacīkammaṃ kusalampi akusalampi, tena vata re vattabbe—“rūpaṃ kusalampi akusalampī”ti.

Ms 32Kv_2413Rūpaṃ kusalākusalantikathā niṭṭhitā.
