Ангуттара Никая

Бала сутта

4.261. Силы

Благословенный сказал: «Монахи, есть эти четыре силы. Какие четыре?

  1. сила усердия,
  2. сила осознанности,
  3. сила сосредоточения,
  4. сила мудрости.

Есть эти четыре силы».
