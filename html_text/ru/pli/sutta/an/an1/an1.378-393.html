Ангуттара Никая

Сутты

1.378

Благословенный сказал: «Монахи, вне сомнений, это один из видов обретений, а именно, быть тем, кто проживает в лесу».

1.379-393

сутты идентичны предыдущей АН 1.378 за некоторыми изменениями:

1.379

быть тем, кто живёт едой, полученной с хождения за подаяниями

1.380

быть тем, кто носит обноски

1.381

быть тем, кто имеет только три одеяния

1.382

быть тем, кто проповедует Дхамму

1.383

быть приверженцем Винаи

1.384

быть тем, кто обладает великой учёностью

1.385

быть тем, кто давно находится в монашестве.

1.386

быть тем, кто обладает подобающими манерами

1.387

быть тем, кто обретает свиту последователей.

1.388

быть тем, кто имеет большую свиту

1.389

быть тем, кто происходит из хорошей семьи

1.390

быть привлекательным

1.391

быть превосходным оратором

1.392

быть тем, кто имеет малое количество желаний

1.393

быть тем, кто имеет хорошее здоровье
