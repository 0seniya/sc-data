Ангуттара Никая

Сутты

2.280

Благословенный сказал: «Монахи, по этим двум причинам Татхагата предписал правила тренировки для своих учеников. По каким двум?

  1. ради благополучия Сангхи и
  2. ради спокойствия Сангхи.

А также по этим двум причинам Татхагата предписал правила тренировки для своих учеников. По каким двум?

  1. ради сдерживания капризных личностей, и
  2. чтобы хорошо себя ведущие монахи могли пребывать в спокойствии.

А также по этим двум причинам Татхагата предписал правила тренировки для своих учеников. По каким двум?

  1. ради сдерживания пятен умственных загрязнений, присущих этой жизни, и
  2. ради рассеивания пятен умственных загрязнений, присущих будущей жизни.

А также по этим двум причинам Татхагата предписал правила тренировки для своих учеников. По каким двум?

  1. ради сдерживания враждебностей, присущих этой жизни, и
  2. ради рассеивания враждебностей, присущих будущей жизни.

А также по этим двум причинам Татхагата предписал правила тренировки для своих учеников. По каким двум?

  1. ради сдерживания изъянов, присущих этой жизни, и
  2. ради рассеивания изъянов, присущих будущей жизни.

А также по этим двум причинам Татхагата предписал правила тренировки для своих учеников. По каким двум?

  1. ради сдерживания опасностей, присущих этой жизни, и
  2. ради рассеивания опасностей, присущих будущей жизни.

А также по этим двум причинам Татхагата предписал правила тренировки для своих учеников. По каким двум?

  1. ради сдерживания неблагих качеств, присущих этой жизни, и
  2. ради рассеивания неблагих качеств, присущих будущей жизни.

А также по этим двум причинам Татхагата предписал правила тренировки для своих учеников. По каким двум?

  1. из сострадания к мирянам, и
  2. прекращения группирований тех, кто обладает порочными желаниями.

А также по этим двум причинам Татхагата предписал правила тренировки для своих учеников. По каким двум?

  1. чтобы те, у кого нет доверия, могли бы обрести доверие, и
  2. ради увеличения доверия тех, у кого уже есть доверие.

А также по этим двум причинам Татхагата предписал правила тренировки для своих учеников. По каким двум?

  1. ради продолжительности благой Дхаммы, и
  2. ради способствования дисциплине.

По этим двум причинам Татхагата предписал правила тренировки для своих учеников».
